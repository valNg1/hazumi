# Metadata — index et métadonnées transverses

Dossier réservé aux **métadonnées transverses** et fichiers d'indexation maintenus au niveau de la bibliothèque (mappings, dictionnaires de tags, exports pour le pipeline RAG).

- Le schéma de métadonnées **par document** est défini dans [`../schema.json`](../schema.json).
- L'index global des documents est [`../index.json`](../index.json).
- Le `.gitkeep` maintient ce dossier versionné tant qu'il est vide.
