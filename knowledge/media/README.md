# Media — actifs binaires

Dossier de stockage des **actifs média** référencés par les documents de la bibliothèque : images, schémas, extraits vidéo/audio, PDF sources, etc.

- Les documents (`.md` / `.json`) des autres dossiers **pointent** vers ces fichiers via le champ `files` ou `url` de leurs métadonnées.
- Nommage en `kebab-case`, sans accents. Sous-dossiers thématiques autorisés.
- Le `.gitkeep` maintient ce dossier versionné tant qu'il est vide.
